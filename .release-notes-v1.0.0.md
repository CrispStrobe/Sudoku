v1.0.0 — First release of CrispSudoku

A polished, ad-free, fully-offline Sudoku game with real depth.

Highlights:

**Four variants**
- Classic, Jigsaw (irregular regions), Sudoku-X (diagonals) and Killer
  (summed cages). Killer is powered by the pure-Dart `dart_csp` constraint
  solver; the rest run on a fast bitmask engine.

**Six grid sizes & four difficulties**
- 4×4, 6×6, 8×8, 9×9, 10×10, 12×12 — Easy to Expert. Every puzzle is
  guaranteed to have exactly one solution.

**Learn as you play**
- "Explain the solve" — a step-by-step walkthrough that names each human
  technique (naked/hidden singles, locked candidates, naked/hidden
  pairs/triples, X-wing).
- "Next logical step" hints explain the next move and why.
- An honest logic difficulty rating based on the hardest technique a board
  actually needs.

**Daily Challenge**
- A new deterministic board each day — the same puzzle for everyone.

**Thoughtful play**
- Pencil-mark notes, unlimited undo, live conflict highlighting, a
  difficulty-scaled mistake limit, per-difficulty hint budgets, themes,
  achievements, streaks and statistics.

**Private by design**
- 100% offline — no account, no ads, no tracking, no data collection.

Free software, open source under the GNU AGPL-3.0.
